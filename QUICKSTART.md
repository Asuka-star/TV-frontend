# 快速启动指南

## 项目概述
这是一个完整的探店平台前端应用，包含动态发布、商铺管理、优惠券秒杀、评论点赞等功能。

## 启动步骤

### 1. 确保后端服务已启动
根据你的 API 文档，需要启动以下服务（或网关）：
- 网关地址：`http://10.21.76.53:8090`（当前配置）

### 2. 启动前端开发服务器

```bash
# 在项目根目录运行
npm run dev
```

### 3. 访问应用
浏览器打开：`http://localhost:5173`

## 主要页面导航

1. **首页** (`/`)
   - 自动重定向到动态广场

2. **动态广场** (`/posts`)
   - 查看所有动态
   - 发布新动态（需登录）
   - 点击动态卡片查看详情

3. **商铺列表** (`/shops`)
   - 浏览所有商铺
   - 搜索商铺
   - 注册新商铺（需登录）

4. **个人中心** (`/profile`)
   - 查看我的订单
   - 管理我的商铺
   - 查看我的关注

## 测试流程建议

1. **注册并登录**
   - 访问 `/register` 注册新账号
   - 使用新账号登录

2. **发布动态**
   - 在动态广场点击"发布动态"
   - 输入标题、内容，可选择上传图片
   - 提交后在列表中查看

3. **注册商铺**
   - 在商铺列表点击"注册商铺"
   - 填写商铺信息
   - 提交后可在"我的商铺"中管理

4. **商铺互动**
   - 浏览商铺列表
   - 点击商铺卡片查看详情
   - 关注、点赞商铺
   - 发表评论

5. **优惠券秒杀**
   - 在商铺详情页查看优惠券
   - 点击"立即抢购"
   - 在"我的订单"中查看订单

6. **订单支付**
   - 在个人中心的"我的订单"
   - 点击"去支付"完成支付
